============ Instruction ============
1. Register your site at Google from here - https://www.google.com/recaptcha/admin

2. Get your Site Key and Secret Key.

3. Open the index.php file. Go to the line no.5 and modify the $secret variable value with your Secret Key.

4. Go to the line no.66 and modify the "data-sitekey" attribute value with your Site Key.

5. Run the index.php file and test the functionality.
